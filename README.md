# Calendar
Interactive web-based calendar built with HTML, CSS, and JavaScript, allowing event creation and dynamic updates.

## Features
- Add, edit, and delete events
- Recurring weekly events
- Course filtering and color coding
- Light and dark mode
- Week view with time grid

## How to Use
Visit the live site: [Live Site](https://iknoorsandhu259.github.io/Calendar/)

No installation required, runs entirely in the browser.

## Local Development
If you want to run it locally:
1. Clone the repository
2. Open `index.html` in any browser

Data is stored in your browser's localStorage and will persist between sessions.
